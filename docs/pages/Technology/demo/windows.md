<li>Sélectionner l'option qui précise que vous n'avez pas de réseau.</li>
<li>Complété la configuration initial.</li>
<li>Ouvrez les paramètre (<code>ms-settings:</code>)</li>

<li>Cliquez 5 fois sur le logo Windows</li>
